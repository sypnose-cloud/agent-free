# agent-free — Free Agents for Claude

Free agents that give Claude real-world capabilities. Works with Claude Desktop and Claude Code.

No API key. No subscription. No catch.

---

## Install

Pick the agent you want and paste in Claude:

| Agent | What it does | Install |
|-------|-------------|---------|
| [browser-agent](./browser-agent) | Navigate any website, click, fill forms, take screenshots | `Install this MCP agent: https://github.com/sypnose-cloud/agent-free/tree/main/browser-agent` |
| [search-agent](./search-agent) | Real-time web search with sources | `Install this MCP agent: https://github.com/sypnose-cloud/agent-free/tree/main/search-agent` |
| [capcut-agent](./capcut-agent) | Create and edit videos in CapCut | `Install this MCP agent: https://github.com/sypnose-cloud/agent-free/tree/main/capcut-agent` |
| [linkedin-agent](./linkedin-agent) | LinkedIn posts, job search, connections | `Install this MCP agent: https://github.com/sypnose-cloud/agent-free/tree/main/linkedin-agent` |

Claude installs each one automatically.
